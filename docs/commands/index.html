Executive#

Information about the Executive of the current nation we're playing as.

important

These commands always vary between MKs, since we might not have an executive in an MK, or the executive doesn't have veto power, in which case all commands but the -executive command are disabled.

The name of this category and all commands will be named after the Executive of the nation we're playing as, but -executive and -exec will always be aliases to these commands.

For example, in MK6, all of these commands were shown as -ministry ... in the -help command. On the other hand, the Nation Bot for the Ottomans in MK7 displayed them as -sultan ....

-executive#

The group command for this category.

Briefly explains what the Bank of Democraciv is, and how to get started.

Aliases

  • -ministry, -minister

Legislature#

Allows the Government to organize legislative sessions and bill & motion submissions

important

These commands always vary between MKs, depending on the Legislature's powers and structure.

The name of this category and all commands will be named after the Legislature of the nation we're playing as, but -legislature and -leg will always be aliases to these commands.

For example, in MK8, all of these commands are shown as -parliament ... in the -help command. On the other hand, the Nation Bot for Rome in MK7 displayed them as -senate ....
